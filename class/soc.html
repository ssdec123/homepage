SoC 설계 교육과정

반도체 SoC 설계의 전체 흐름을 실습 중심으로 학습하는 전문 교육 프로그램

교육 개요

SoC 설계 교육과정은 RTL부터 합성, P&R, 타이밍 분석, ECO까지 실제 SoC 설계 플로우를 실습 중심으로 다루는 심화 과정입니다. 반도체 칩 설계 전반의 흐름을 기반으로 실무 능력을 갖춘 SoC 엔지니어 양성을 목표로 합니다.

이 과정의 특징

Full SoC Flow 실습

RTL → 합성 → APR → STA까지 전 과정 실습

최신 공정 기반 이해

5nm/3nm 공정 기반 SoC 설계 이해

EDA Tool 실습

Synopsys / Cadence 주요 툴 실습

타이밍 분석 & ECO

실제 칩 설계에서 사용하는 타이밍 분석 프로세스 학습

커리큘럼

1~2주차: RTL 설계 기초 / Verilog

3~4주차: 합성(Synthesis) / Constraint 적용

5~6주차: APR (Placement & Routing)

7~8주차: 시그널 무결성(SI) / 전력 분석

9~10주차: 타이밍 분석(STA) / ECO

11~12주차: Mini SoC Project

지원 안내

  • 지원 대상: SoC 설계 진로 희망자 또는 기초 설계 경험 보유자
